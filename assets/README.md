# Assets

This directory contains images and screenshots for the WiseRate project.

## Contents

- `screenshot.png` — Main app screenshot
- `ios-app.png` — iOS app screenshot
- `web-app.png` — Web app screenshot

## Specifications

| Image | Size | Format | Description |
|-------|------|--------|-------------|
| screenshot.png | 1280×720 | PNG | Main app view |
| ios-app.png | 390×844 | PNG | iOS app view |
| web-app.png | 1280×720 | PNG | Web app view |

## How to Capture

1. Open the app on iOS or Web
2. Navigate to the main screen
3. Take a screenshot
4. Save the image in this directory
5. Update the README.md to reference the new image

## Optimization

- Use PNG format for screenshots
- Keep images under 1MB
- Compress with [TinyPNG](https://tinypng.com) if needed
